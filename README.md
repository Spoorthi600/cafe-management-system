The Cafe Management System is a full-stack MERN application built to manage cafe operations efficiently. 
It features dynamic menu management, real-time order tracking, secure user authentication, and a responsive admin dashboard. 
The frontend is developed in React.js, while the backend uses Node.js, Express, and MongoDB. 
To run the project, clone the repo, install dependencies in both `client` and `server` folders, and start the development server.
